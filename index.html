Bouncybox

A jQuery plug-in that lets you create a dialog that will bounce its way in and out!

Welcome to BouncyBox!


Bouncy box is a light-weight jQuery plug-in which enables you to create contents that will have a bouncing animation as it enters and exits the screen.

In other words, they will go like bounce! and bounce! and bounce like a ball!

Methods

  • bouncyBoxRegister()
  • bouncyBox()
  • bouncyBoxDismiss()

bouncyBoxRegister()

Description
This method initializes the element(s) to be used with the bouncyBox() method. This method simply reposition the target element(s) to the initial position when bouncyBox() method is used. It is best to invoke this method to the target element(s) right after the page has been loaded(ready()).

Syntax
$(selector).bouncyBoxRegister()

Parameters
none

Return Value
Returns a jQuery object

Example
$("#selector1,#selector2").bouncyBoxRegister();

bouncyBox()

Description
Performs the bounce animation to the target element.

Syntax
$(selector).bouncyBox([speed])
$(selector).bouncyBox([callback])
$(selector).bouncyBox([speed,callback])

Parameters
speed (integer)- Sets the speed or duration of the animation. The lower the value, the faster the animation.
callback (function) - A function that will be invoked right after the animation has completed.

Return Value
Returns a jQuery object

Examples
Example 1: No parameter
$("#selector1").bouncyBox();
Example 2: Speed parameter
$("#selector1").bouncyBox(500);
Example 3: Callback function parameter 

$("#selector1").bouncyBox(function(){
    alert("Done!");
})

Example 4:Speed and Callback parameter 

$("#selector1").bouncyBox(500,function(){
    alert("Done!");
})

bouncyBoxDismiss()

Description
Hides back the target element by bouncing it away. This is the opposite to bouncyBox() method.

Syntax
$(selector).bouncyBoxDissmiss([speed])

Parameters
speed (integer)- Sets the speed or duration of the animation. The lower the value, the faster the animation.

Return Value
Returns a jQuery object

Example
$("#selector1").bouncyBoxDismiss();
